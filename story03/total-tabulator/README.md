# Total Tabulator

| Expected file |
| ------------- |
| `total-tabulator.sql` |

Perform table-wide calculations.

## Instructions:

Using the `products` table, write a query to calculate the total value of all products in stock. The total value is the sum of `price * quantity` for all products.
