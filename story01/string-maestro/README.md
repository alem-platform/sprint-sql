# String Maestro

| Expected file      |
| ------------------ |
| `string-maestro.sql` |

Manipulate strings like a maestro. Use string functions to transform your text data.

### Instructions:

- Display `first_name`, `last_name` and a new column `email_domain` that extracts the domain part of the `email` (the part after `@`).
