# Cross Them All

| Expected file |
| ------------- |
| `cross-them-all.sql` |

`CROSS` them all!

## Instructions:

Create a list of all possible combinations of media types (as `MediaTypeName`) and genres (as `GenreName`). The result should pair every media type with every genre and order by `GenreName` in descending order.
